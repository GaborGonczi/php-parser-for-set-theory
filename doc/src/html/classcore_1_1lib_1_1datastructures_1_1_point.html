core\lib\datastructures\Point::jsonSerialize ( )

Serializes the point object to JSON format.

This function implements the JsonSerializable interface, which allows the point object to be serialized by the json_encode function. The function returns an associative array with three keys: 'name', 'x', and 'y'. The values of these keys are the name, x-coordinate, and y-coordinate of the point object, respectively.

Returns
array An associative array with three keys: 'name', 'x', and 'y'.
