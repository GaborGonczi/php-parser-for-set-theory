app\server\classes\Auth::loginHandle ( )

Handle the login form submission.

This method validates the username and password from the POST data and checks if they match a record in the users table. If they do, it creates a User object and stores it in the session and cookie. It also updates the first login, last login, and modified at fields of the user record. If they don't, it sets an error message in the session. It then redirects to the index page.

app\server\classes\Auth::registerHandle ( )

Handle the register form submission.

This method validates the username and password from the POST data and checks if they are available and match. If they do, it creates a new User object and inserts it into the users table. It then redirects to the index page. If they don't, it sets an error message in the session.
