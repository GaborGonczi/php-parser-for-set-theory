Checks if a point is inside a quadrangle.

This function takes the coordinates of a point and the four vertices of a quadrangle as arguments and returns a boolean value indicating whether the point lies inside the quadrangle or not. The function uses the winding number algorithm to determine the point-in-polygon test, which involves calculating the cross product of the vectors and the position of the point relative to the edges of the quadrangle. The function also uses the @codeCoverageIgnore annotation to exclude it from code coverage analysis.
