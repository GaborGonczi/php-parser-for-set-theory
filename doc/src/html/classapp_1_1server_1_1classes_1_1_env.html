app\server\classes\Env::load ( )

Load the environment variables from the .env file.

This method reads the .env file line by line and parses the name and value of each environment variable. It ignores the lines that start with # or are empty. It also checks if the environment variable is already set before overriding it. It sets the environment variables using putenv, $_ENV, and $_SERVER arrays. It also checks if the file is readable and throws an exception if it is not.

Exceptions

RuntimeException If the file is not readable.
